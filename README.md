课程介绍
长期以来机器学习很多教材描述晦涩难懂，大量专业术语和数学公式让学生望而止步。生活中机器学习就在我们身边，谷歌，百度，Facebook，今日头条都运用大量机器学习算法，实现智能推荐功能。我用大量精美配图来解释机器学习算法原理。

此课程收录了大量MIT麻省理工学院知名教授的机器学习和深度学习课程系列，各位学员记得报名此课程哟。
